## Installation

```bash
$ git clone https://github.com/LeonardoFariaOliveira/User-Microservice
$ cd userarea-service
$ yarn
```

## Running the app

```bash
# watch mode on development
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
